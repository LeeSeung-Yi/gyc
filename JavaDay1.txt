1.1 프로그래밍 언어와 자바

0) 프로그래밍 언어(자바, c, c++,c#, 파이썬) 
- 고급언어: 사람이 이해할 수 있는 언어, 컴파일 과정 통해 컴퓨터가 이해 할 수 있는 언어로 변환
- 저급 언어: 기계어, 어셈블리어

프로그램 언어로 소스파일 작성- 소스파일- 컴파일러- 기계어 파일- 내가 아는 언어로 실행


1) 자바
모든 운영체제 가능, 웹사이트, 개발안드로이드 앱 개발, 데스크톱 앱 개발
- 객체 지향 프로그래밍: 부품 만들고 객체(부품)들을 연결해서 더 큰 프로그램 생성
- 메모리 자동 관리
- 풍부한 무료 라이브러리로 개발 기간 단축
자바(JDK) 설치하고 나면 다른 프로그램들이 이용할 수 있도록 JAVA_HOME으로 환경 변수 생성,
PATH통해 수정,  
<순서>
환경변수
새로만들기
새 시스템 변수, 변수이름: JAVA-HOME, 변수값: JDK설치 경로 입력
JDK설치 경로인 BIN디렉토리- 안 명령어는 컴파일 실행할 때 사용,, BIN디렉토리에 있지만 어떤 위치에서도 사용하도록 환경변수에서 편집
PATH로 경로 추가
새로만들기- 환경변수 값 입력- 위로 이동(명령어 사용 위해)


2) JDK설치 후 자바 언어로 작성한 소스파일 만들고 컴파일(컴퓨터 언어로 변환)
자바 소스 파일 확장명 .java(텍스트 파일) : 내가 자바로 언어 만들고 컴퓨터 언어로 변환

- 바이트코드 파일: 소스파일을 작성한 후에 컴파일
자바소스파일(Hellojava)작성- 자바 명령어 실행(javac명령어로 소스파일 컴파일)- 바이트코드 파일 생성(.class)

- 자바 가상 머신
바이트 코드 파일을 특정 운영체제가 이해하는 기계어로 번역하고 실행시키는 명령어: JAVA
